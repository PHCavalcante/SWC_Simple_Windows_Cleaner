# Instaling

```
git clone https://github.com/PHCavalcante/SWC_Simple_Windows_Cleaner.git
```

Installing dependencies:

```
pip install requests
pip install customtkinter
pip install PIL
```

Run the program:
```
python main.py
or
python3 main.py
```
