# Instaling

```
git clone https://github.com/PHCavalcante/SWC_Simple_Windows_Cleaner.git
```

Installing dependencies:

```
pip install 
pip install requests
pip install customtkinter
pip instal PIL
```

Run the program:
```
python SWC.py
or
python3 SWC.py
```
